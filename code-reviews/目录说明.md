# Cline 项目目录结构说明

## 项目概述

**Cline** 是一个功能强大的 AI 编程助手，作为 VS Code 扩展运行。它能够使用命令行（CLI）和编辑器进行自主编码，支持创建/编辑文件、执行命令、使用浏览器等功能。项目采用 TypeScript 开发，支持多种 AI 模型提供商（如 Anthropic Claude、OpenAI、Google Gemini 等）。

## 技术栈总结

- **语言**: TypeScript
- **运行环境**: VS Code Extension、Node.js CLI
- **前端框架**: React + Vite
- **UI 组件**: shadcn/ui + Tailwind CSS
- **构建工具**: esbuild、Vite
- **测试框架**: Mocha、Vitest、Playwright
- **代码规范**: Biome (Lint + Format)
- **通信协议**: gRPC、Protocol Buffers
- **AI 集成**: 40+ AI 模型提供商
